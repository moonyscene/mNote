    Print 배경인쇄

    [Internet Explorer] 배경색과 이미지를 인쇄하기 위해서는 도구 메뉴의 인터넷 옵션(O)을 클릭하면, 고급 탭의 설정 항목 중에 인쇄에서 배경색 및 이미지 인쇄에 체크를 하면 된다.
[Firefox3.x] 배경인쇄는 파일(F) 메뉴의 페이지 설정(U)에서 지정할 수 있음.
[Opera 9.x] 페이지의 배경인쇄는 파일(F) 메뉴의 페이지 설정(T)에서 지정할 수 있음.
[Safari 3.x] 배경인쇄 옵션을 찾을 수 없어 배경이 전부 인쇄 되지 않으며, 특이하게 흰색 글씨는 회색으로 출력 됨.(좌)
[Chrome] 인쇄 미리보기 기능이 없으며, 배경이 인쇄되지 않음(우)

    C:\Windows\System32\drivers\etc\hosts 파일 내용이 권한 때문에 수정이 안되면?

    다른 폴더에 복제해서 파일 내용 수정한다.
기존 hosts 파일 이름을 host.-2014MMDD 형태로 백업한다. (파일 이름은 수정되더라.)
수정한 파일을 C:\Windows\System32\drivers\etc\ 로 복사한다.
