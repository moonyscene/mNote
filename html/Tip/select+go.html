<select>(목록상자) [이동]버튼

Revision : 2010.05.04

서식제목
웹 접근성 : 위와 같은 형태의 콘텐츠를 자바스크립트 없이도 작동하도록 하는 소스
(X)HTML, JavaScript, ASP, PHP, JSP
(X)HTML
<form action="/share/ui/gouri.jsp" target="_blank"> <fieldset><legend>서식제목</legend> <select name="uri" class="select" title="바로가기"> <optgroup label="옵션그룹이름"> <option value="">선택 후 이동을 클릭</option> <option value="http://www.wah.or.kr/">웹 접근성 연구소</option> </optgroup> </select> <input type="submit" value="이동" class="button" onclick="goSelect(this.form,'_blank'); return false;" title="새 창에서 열림" /> </fieldset> </form>

gouri.* 가 있으면 onclick="goSelect(this.form,'_blank'); return false;" 삭제하여도 되지만,
<option value=""> 의 경우 빈 새 창을 보여주므로 그냥 두는 것이 좋겠다.
팝업 창처럼 새 창의 윈도우 모양을 제어하려면 goSelect()를 수정하여 사용.

※ 이동버튼은 아래 3가지 중 상황에 따라 사용하세요.
<input type="submit" value="" class="button" /> <input type="image" src="" alt="이동" class="image" /> <button type="submit" class="button">이동</button>
JavaScript
<script> //선택상자바로가기 function goSelect(form,target){ var myindex=form.uri.selectedIndex; myUri=form.uri.options[myindex].value; if(myUri!=""){window.open(myUri,target);} } </script>
ASP : /share/ui/gouri.asp
<% Dim uri uri = Request("uri") IF uri <> "" Then Response.Redirect uri Response.end End IF %>
JSP : /share/ui/gouri.jsp
<%@ page contentType="text/html; charset=utf-8" %> <% String uri = request.getParameter("uri"); if(uri!=null && !uri.equals("")) response.sendRedirect(uri); %>
PHP : /share/ui/gouri.php
<? if($uri && $uri!="") header("Location:".$uri); ?>
