    IE7과 동일

    • filter: 속성 적용 가능.
      background: #0ff; opacity: 0.5; filter: alpha(opacity=50);

    Bug

    • display: none; display: block; 동작시 가끔 늘어나고 줄어드는 공간이 틀어져 겹쳐지는 문제 발생.
    • display: inline-block; 오류.. 간헐적으로 발생하여 정확한 증상은 파악하기 힘듦.
    • iframe src 와 부모 프레임의 인코딩이 utf-8 과 utf-8 로 서로 다를 때 a 요소에 마우스 올려 포커싱하면 화면 전체의 텍스트가 1px 움직인다.
    • overflow: auto; 또는 textarea 의 스크롤 동작 오류. 내부 영역을 선택하거나 입력하거나 하는 동작시 스크롤이 다른 위치로 이동되어 버린다. overflow: auto; 의 경우 display 스타일시트 속성으로 보였다 감추는 동작을 가지는 콘텐츠(예:js로구현한콤보상자)에서 발생하였는데.. visibility 로 하니 버그 현상은 사라졌다.
