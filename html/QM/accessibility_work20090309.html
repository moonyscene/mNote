작업 대상 범위

  • 웹사이트 : [표1] 진해, 창녕, 사천, 산청, 의령 (본청, 회원로그인, 열린시장(군수)실, 참여시스템, 문화관광)
    .. 과업에 포함되지 않더라도 본청에서 해당 링크가 있으면 점검 필요.
  • 웹페이지 : 18개 지표에 포함되는 콘텐츠가 있는 모든 웹페이지 (플래시, 동영상, 그래픽, 서식 등)
  • 실작업자 : [표1] 디자인코더, 프로그래머

사이트별 작업 방법


사전 심사 대비 ~3/11이전에 작업완료 필요!

  • 본청 대체텍스트 제공 (자동, 90%이상) - KADO-Wah 자동 평가
    ※ 주의 : 본청 하위 폴더에 존재하는 사이트로 인하여 90%이상 안나오면 하위 도메인으로 변경 등의 대책을 클라이언트에 제시하자!!
  • 테이블 의미태그 (수동, 70%이상) - 데이터 테이블은 summary, caption, h1~h6 중 하나라도 제공, thead, tbody, th 태그 사용.
  • 온라인 서식 레이블 (수동, 90%이상) - label의 for 속성과 id값 연결.

사전 심사에서 부적합 받으면 전문가 심사도 진행하지 않고 평가 결과 보고서도 받을 수 없으므로 반드시 작업하도록 하세요!!

전문가 심사 대비

  • 장애인 웹 접근성 국가표준 준수 기술 가이드라인 18개 지표 모두 준수 기준에 적합하도록 작업한다.
위 지표에 언급되지 않은 것( scope="col" 등)도 분명 접근성을 개선하는데 필요하지만, 이번 품질마크 대비 작업에서는 신경쓰지 않아도 됩니다.
