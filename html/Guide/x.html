레이아웃 - id/class 선택자 선언 규약 | Naver Naming Guidelines 참고하여 수정

#container : 콘테이너 - 페이지 전체(#head, #wrap, #foot) 담기.
#head : 머리
　h1#logo : 사이트 로고
　#globalmenu : (Site Menu) - 홈, 로그인, 사이트맵, 외국어 등
　#topmenu : (Local Menu) - 주메뉴
　#search : 통합검색 폼
#wrap : 랩 - 옆구리(#sidebar)와 몸통(#body)을 감싸기.
#visual : 비쥬얼 - 보통 #wrap 밖에 있지만, 랩 밑에 깔아서 안에 있는 것처럼 보입니다.
#sidebar : 옆구리
#sidemenu : 부메뉴
#sm1 : 보충 메뉴들
.colgroup : 컬럼 그룹핑 (.colgroup 와 #aside 는 거의 안씀)
#body : 몸통
#body_head :
　#location : 문서 경로(현재 위치)
　#body_title : 본문 제목 <h1>
#body_content : 본문 내용
　본문 소제목 1차 (본문 2차 메뉴) <h2>
#body_foot :
　#charge : 본문 책임자, 최종 수정일
　#bodyutil : 본문 유틸리티
　#satisfaction : 만족도 조사
#aside : 곁가지
#am1 : 딴 메뉴

#foot : 다리
　#siteutil : 사이트 유틸
　#siteguide : 사이트 안내 및 정책
　#authorinfo : 작성자 정보 (연락처 및 저작권)
#wing : 날개
　#wm1 : 퀵메뉴
　.gotop : 위로

기본 /폴더/파일 설명

구분 /폴더/파일 설명 비고
/ /index.html 루트 폴더에는 메인페이지로 가는 기본문서 1개만 둔다
폴더 /01alphabet/ ~ /06alphabet/ 주메뉴 페이지 폴더명은 2자리순번과 1차메뉴영문명 을 조합해서 만든다.
예) 전자민원 - /01appeal/
/data/ 다운로드 자료, 멀티미디어 파일들 : hwp, doc, pdf, ppt, xls, wmv 등..
필요에 따라 하위 폴더를 자료 종류별 혹은 주메뉴 분류별로 생성한다.
※주의 : /img/ 에 첨부 파일 자료를 절대 넣지마세요!!
/etc/ 기타, 추가페이지 : 주메뉴에 분류되지 않는 페이지 들, sitemap.jsp 등
/flash/ 모든 플래시 파일.. swf, 퍼블리시한 html 파일,
/img/ 모든 이미지파일은 이 폴더 아래에둔다. 필요에따라 하위폴더 생성해서 정리
※주의 : 웹페이지 디자인 이미지 외에 첨부 파일 자료를 절대 넣지마세요!!
/main/ 메인페이지 및 메인에만 사용하는 인클루드 파일 등
메인페이지는 ~/main/ 폴더 까지만으로도 페이지가 열리도록 기본문서를 사용한다.
/share/ 공유폴더
이미지 /img/01/ ~ /img/06/ 이미지 폴더명은 1차메뉴 순번(2자리)으로 한다. - 이미지 경로가 너무 길어지는거 방지.
/img/all/ 본문에 공용으로 사용하는 이미지.. 불릿 등
/img/banner/ 배너이미지 - 메인과 서브 배너모음 페이지 등에서 공통사용
/img/board/ 멀티게시판용 이미지
/img/etc/ /etc/ 문서용 이미지 폴더
/img/inc/ 인클루드 파일에 사용되는 이미지.. 로고, 주메뉴, 비쥬얼 등
/img/inc/side/ 부메뉴를 이미지로 만들 때 사용
/img/inc/title/ 본문 타이틀 이미지용
/img/main/ 메인페이지에 사용되는 이미지
/img/main/picpr/ 메인 이지미홍보(팝업존)에 사용되는 이미지
스타일시트 /share/css/ 모든 스타일시트 파일들
200.08이전 /share/css/all.css 스타일시트 @import 및 모든페이지 공용 스타일시트 정의
200.08이전 /share/css/base.css 기본 태그에 정의한 스타일시트 img, form { margin: 0; border: 0; } 등.. 수정 거의 필요 없음
/share/css/board.css 멀티게시판용
/share/css/content.css 본문 콘텐츠용 개별 스타일시트 정의
200.08이전 /share/css/layout.css 전체 레이아웃용 : head, foot, body_head, body_foot, wing, 사이트맵 등
/share/css/main.css 메인 페이지용
200.08이전 /share/css/template.css 사이트 템플릿 스타일시트 정의. <h1>~<h6> 등
인클루드 /share/inc/ 레이아웃용 인클루드 파일들 ** 본문에서 인클루드가 필요한 경우 해당 폴더에서 만듦
/share/inc/body_foot.jsp 본문 푸트 영역 : 담당자, 업데이트 날짜, 인쇄버튼, 뒤로버튼, 위로버튼 등 
/share/inc/body_head.jsp 본문 헤드 영역 : 3차(2차)타이틀, 현재 위치 등
/share/inc/foot.jsp 푸트 영역 : 하단 로그, 하단 메뉴, 주소, 연락처, 카피라이트, 카운터 등
/share/inc/head.jsp 헤드 영역 : 로그, 글로벌 메뉴, 주메뉴(1,2차) 등
/share/inc/html_head.jsp <head></head>태그 사이에 공통으로 들어가는 것
/share/inc/menu_all.jsp 메뉴명, 메뉴링크, 메뉴 onclick 스크립트를 배열로 정의해 놓음. 4차메뉴까지를 기본으로 정의한다.
/share/inc/openwin.jsp 새창 코드 샘플
/share/inc/printpage.html 프린트 스크립트로 불러오는 공용 프린트페이지
/share/inc/sidebar.jsp 사이드바 영역 : 1차타이틀, 부메뉴(2,3차), 서브 공통 메뉴 등
/share/inc/wing.jsp 날개 메뉴 영역 : 퀵메뉴, 서브 공통 메뉴
자바스크립트 /share/js/ 모든 자바스크립트 파일들
/share/js/all.js 모든페이지에 공용으로 쓰는 자바스크립트
게시판 /share/board/ 게시판 코드 샘플
/share/board/board.jsp 게시판 전체 포함한 페이지 
/share/board/board_inc.jsp 게시판 전문, 콘텐츠 메뉴 샘플
/share/board/comment.jsp 댓글
/share/board/find.jsp 아이디/비밀번호 찾기
/share/board/list.jsp 목록
/share/board/list_gallery.jsp 목록 - 갤러리형
/share/board/list_webzine.jsp 목록 - 웹진형
/share/board/login.jsp 로긴폼
/share/board/view.jsp 내용
/share/board/write.jsp 쓰기폼

네이밍 규칙 (/폴더/파일명 명명규칙)

구분 예시 규칙 비고
기본 명명규칙 모든 폴더, 파일명은 소문자로 만든다.
긴 이름은 소문자_소문자 형태로 만든다.
00_inc.jsp 인클루드 파일 확장자도 개발언어 확장자를 사용한다.
링크주소, 이미지경로는 절대경로를 사용하고 부득이한 경우 상대경로를 사용한다.
폴더/웹문서명 /1차메뉴영문명/2차메뉴영문명_3차메뉴영문명.개발언어확장자
작업 간소화를 위해 2자리 순번으로 파일명을 생성할 수도 있다.
/01appeal/index.jsp 1차메뉴의 서브메인(인덱스)페이지 : 기본문서로 만든다.
/01appeal/01.jsp 1차메뉴1번째_2차메뉴1번째
/01appeal/02_01.jsp 1차메뉴1번째_2차메뉴2번째_3차메뉴1번째페이지
/01appeal/01_inc.jsp 1차메뉴1번째_2차메뉴1번째의 인클루드파일 (켄텐츠안 메뉴 등을 만들 때)
이미지명 이미지파일명은 작업자가 알아서 명명
/img/01/01_01_p01.gif 1차메뉴1번째_2차메뉴1번째_3차메뉴1번째 웹문서에 1번째로 사용한 이미지(picture)
/img/01/01_01_p02.gif 1차메뉴1번째_2차메뉴1번째_3차메뉴1번째 웹문서에 2번째로 사용한 이미지(picture)
추가 폴더 /pop/ 팝업용 폴더
/pop/popup.js 팝업용 자바스크립트
