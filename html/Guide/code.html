웹 표준

전자정부서비스 호환성 준수지침 (행정안전부, 2009.8~2012.10)

  1. 문자인코딩 : UTF-8 을 사용하는 것을 원칙으로 한다. 기술 제약, 서비스 특성으로 인해 다른 방식을 사용한다면 UTF-8 에 준하는 수준으로 한글을 처리할 수 있도록 조치하여야 한다.
  2. HTML : W3C Markup Validator 통과
  3. CSS : W3C CSS 2.1 또는 CSS 3 로 구현. W3C CSS Validator 통과
  4. JavaScript : W3C DOM Level 2 또는 Level 3 또는 ECMA-262 3rd
  5. 신기술 : 웹사이트 호환성 확보가 가능한 수준에서 HTML5 등의 신기술을 사용할 수 있다.
  6. 모바일 : 모바일 웹 방식으로 제공. 단, 기술 제약, 비용 문제 있으면 다른 방식으로 제공할 수 있다.
