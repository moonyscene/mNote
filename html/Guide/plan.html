  1. 콘텐츠 구조는 클라이언트명_사이트명_콘텐츠 구조 v1.0.xls 처럼 파일명과 버전을 명시하고 엑셀파일로 통일해주세요.
  2. 콘텐츠 구조에는 자료 및 구현 방안 파악이 용이하도록 상세히 기록해주세요.
  3. 콘텐츠 구조에 모든 페이지(상·하·좌·우 공통메뉴 링크된 모든것)를 나열해주세요.
    (로그인, 회원가입, 로그아웃, 회원정보수정, MY페이지, 사이트맵, 뷰어다운로드, 개인정보 보호정책, 홈페이지 의견수렴 등)
    유사한 메뉴끼리 반드시 묶어주세요. "이용안내", "기타정보", "회원관리" 등으로..
  4. 최종PSD파일과 콘텐츠 구조엑셀 파일의 메뉴명을 동일하게 맞춰주세요.
  5. 콘텐츠 번호(예: c1.1.1)에 오류가 없도록 최종본 넘기기 전에 다시 한번 반드시 확인해주세요.
  6. 콘텐츠 구조엑셀에서 서브 인덱스(혹은 인트로) 페이지가 있으면 하위 콘텐츠 메뉴는 다음 행부터 표시해주세요.
  7. 콘텐츠 구조에 메인콘텐츠 내용도 표시하고 해당자료, 링크 추가페이지 내용 등도 정리해주세요. (가능하면 1줄로 정리)
    • 메인 콘텐츠 레이블과 메뉴 구조의 이름이 통일되어 있다면 외부 링크만 정리해주시고,
    • 통일이 안되어 있거나, 복잡한 경우에는 콘텐츠 구조 아래 메인 콘텐츠를 나열하든가 별도 시트 만들어서 정리해주시기 바랍니다.
  8. 콘텐츠 구조가 수정되면 파일을 따로 만드시고(버전 표시), 이전과 수정된 내용은 글자색 등으로 구분해서 표시해주세요.
    (표시가 없을 경우, 수정 부분 파악 곤란함. 엑셀파일 공유해서 버전 관리 함께 한다..)
  9. 콘텐츠 구조 엑셀파일의 자료란에는 자료 위치만 표시하고 내용(수정사항)은 별도 자료로 만들어주세요.
  10. 자료
    • 파일로 있는것은 파일명을 명시해주세요.
    • 기존 홈페이지에서 내용 가져올 때는 기존 웹 URL을 명시하시고 활용정도를 알려주세요.
      (링크, 내용 그대로, 화면구성 참고, 디자인만 변경, 일부내용 수정(콘텐츠 구조에 수정내용 적지말고 파일로 따로 정리))
    • 프린트 자료 입력시에는 프린트 자료 입력이라고 해주시고, 프린트 자료에는 해당 콘텐츠 번호(예: c1.1.1)와 메뉴명(예: 무인민원창구)을 적어주세요.
    • 파워포인트 등으로 정리한 윈도우설명서(화면 구현 방법) 등이 있을 때는 콘텐츠 구조 엑셀파일에 무슨 자료 참고라고 명시해주세요.
    • 메인, 서브 비쥬얼에 들어갈 사진, 캐치프레이즈 등을 클라이언트와 협의해서 작업 전에 알려주세요.
    • 사진, 이미지 자료 제공시 내용페이지와 연계해서 클라이언트가 요구하는 이미지를 알기 쉽게 정리해서 주세요.
    • 내용페이지, 게시판 등에 들어갈 전문을 미리 정해서 알려주세요.
  11. 업무요청(추가 자료 전달, 수정)시에는 말로 전달(기억 못함)하지 마시고 가능한 글(알림이 쪽지 또는 파일)로 주세요.
  12. 작업 시작 전 자료 주실 때는 1,2차 메뉴별로 폴더를 만들어주시고, 추가 자료는 그 폴더에서 찾아가라고 하지 마시고 직접 보내주세요.
  13. 작업 요청시 가급적 모든 자료를 준비해주세요. (일정에 맞게 자료 준비를 해주세요.)
  14. 작업량을 파악해서 정리해주세요. (과업범위, 전체 페이지수, 플래시, 특화콘텐츠 등..)
    특히, 과업지시서에 없는 본청의 마이크로 사이트 등 포함 여부 파악 필요
  15. 프로젝트 시작전에 다음 사항을 공유합시다. (기본 공유 파일)
    과업지시서 문서파일, 콘텐츠 구조 엑셀파일, 프로젝트 기본정보 파일 사업명/사업기간/참여인원/주요일정/과업범위/구축 사이트 정보(개발언어)
  16. 론칭 후 사용할 웹주소 (운영 도메인)를 확정해서 알려주세요.
  17. 메뉴명들 띄어쓰기 보면 제각각인데.. 규칙을 정해서 해주셨으면 합니다.
    메뉴명을 고유명사로 생각하여 붙여서 보내주시는 경우가 많은데..
    맞춤법에 맞춰서 주시면 감사하겠습니다.(가급적 붙이지 말고 띄어쓰기 해서..)
  18. 콘텐츠가 동적이거나 설명이 필요할 때, 스토리보드에 예상할 수 있는 모든 경우(최소~최대 내용)를 고려하여 구현 설명을 넣어주세요.
    갑과 협의하여 기획안을 확정해주시면, 작업 후 변경 작업을 최소화할 수 있습니다.

외국어 번역 자료 준비시 유의사항

  1. 기본적으로 3차메뉴까지는 c1.1.1 이런식으로 일련번호를 붙이고, 페이지가 구분되면 반드시 하위 일련번호를 붙이세요.
  2. 메인 콘텐츠, 이미지, 플래시 내용, 캐치프레이즈, 지도, 설명문구, 버튼명, 대체텍스트 등을 누락하지 말고 정리해서 한번에 번역해서 주세요.
  3. 중복된 번역 내용이 있어도, 어디 내용 참조라고 하지 말고 해당 메뉴 위치에 내용 같이 넣어서 주세요.
  4. 한글원본과 번역내용의 문단모양을 통일해주세요. (줄바꿈 등..)
    번역 페이지수 줄이기 위해서 붙여놓은거 다시 원상태로 돌려주세요. 줄바꿈, 테이블 모양 등.(br 태그 구분 필요없어요.)
    테이블 데이터를 텍스트로 나열해서 번역했으면, 자료 넘겨주실때 테이블 원래 모양대로 만들어서 주세요.
  5. 키보드에 없는 특수 문자는 가급적 사용하지 마세요. ○ 등등..
  6. 각 언어별로 따로 파일을 만들어 주세요..
    간혹 한글원본과 번역이 같이 들어있는 경우가 있는데.. 삼가해주세요..
  7. 번역업체에서 번역자료 받을 때 워드문서와 함께 아래 해당언어로 인코딩된 인터넷문서도 함께 받으세요.

접근성만 개편 과업 자료 준비

  1. 과업지시서, 프로젝트 기본정보, 콘텐츠 구조 엑셀 파일
  2. 디자인 소스 : 현재 웹사이트와 동일한 최종본 포토샵 파일(*.psd), 플래시 소스파일(*.fla)
  3. 대체텍스트 : 약도, 흐름도 등 내용 파악이 어려운 이미지의 설명글, 동영상(플래시 홍보 영상 포함), 음성 콘텐츠의 음성 내용. 음성이 없으면 화면 내용을 설명하는 화면해설 내용 원고(대본) 등..
  4. 외국어 번역본 텍스트 : 메뉴명, 메인 콘텐츠, 서브 내용이 현재 사이트와 완전히 매칭되어 정리된 형태로 준비.
  5. FTP : 기존(현재 운영중인) 웹사이트별 /폴더/ [URI] 을 정리해주세요.
