Website Template Coding Pre-Checklist

템플릿(기본 레이아웃) 코딩前 체크리스트

  • 본 체크리스트는 템플릿 코딩을 효율적이고 원활하게 수행하기 위하여 필요한 점검 사항입니다. (항목 의미 파악이 안되면 물어보세요!)
  • 담당자별로 모든 점검 항목에 충실히 답변·조치해주시기 바랍니다. (확인·조치 없는 거짓 답변은 나빠요~)
  • 즉시 답변할 수 없는 경우에는 답변 가능(체크리스트 작성 및 조치 완료) 시점을 먼저 알려주세요. (보류하고 다른 작업할지?)
  • 보완 조치가 완료되면 별도의 체크리스트 답변 파일은 필요하지 않습니다. (검토할 파일 수가 많아지는 것은 원하지 않습니다.)
    PMS 와 IA, Design, Storyboard 파일에 반영되어 코딩이 끊김 없이 진행 가능하도록 해주시면 됩니다.
  • 설계 내용을 텍스트로 표현하기 힘든 경우.. 간략히 설명을 적고 오디오로 상세 설명을 해주세요.
  • 콘텐츠가 단순 정보인지 링크인지 알 수 있도록 표시주세요.
  • Reference / Benchmark Site 요청하자! _기획, 디자인 넘겨주실 때.. 항상.. 벤치마킹 사이트 있으면 알려주세요!!

    3. Design 과 IA 동일 ▶정리(콘텐츠넘버 표기) 불필요
    4. 맞춤법, 띄어쓰기 문제여부? ▶ IA 와 Design 수정
    5. IA 언급 없으나, 필수 디자인 누락여부? ▶ Design 수정
