/share/css/*.css 최적화

수정 파일 수정 설명 기존 파일 설명
all.css 공용. 태그 기본, 레이아웃 스타일 기존 template.css 일부(태그 기본 스타일),
기존 layout.css 일부(#head, #foot, #wing 자식 요소와 #openwin).
기존에는 @import 도 사용했지만, 앞으로는 link 로만 외부CSS 파일 참조하기로 함.
main.css 메인용. 레이아웃 스타일 보정, 콘텐츠 스타일 기존 main.css 와 전부,
기존 template.css 일부(메인에서도 사용하는 스타일).
sub.css 서브용. 레이아웃 스타일, 본문 스타일 템플릿.
 기존 layout.css 일부(#sidebar, #body_head, #body_foot 자식 요소),
기존 template.css 대분분.
board.css 게시판용. 기존 board.css 전부,
기존 template.css 일부(폼,링크,페이징,달력).
content.css 서브 본문용. 기존 content.css 와 동일.
print.css 인쇄용. 기존 print.css 와 동일.

횡성 메인 페이지의 경우..
요청18(css 9 + js 9) → 요청2 (main.css + main.js)
css 전체 용량 144KB → 35KB
js 전체 용량 46KB → 12KB
평균 속도 1/2 이상 줄임. (더불어 다른 최적화 작업도 한 후 결과임)
