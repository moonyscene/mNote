    Outline

    1. XE는 회원가입, 게시판, CMS(Content Management System) 기능을 갖춘 오픈소스.
    2. 한국적 취향에 가장 잘 특성화된 프로그램으로 손쉽게 웹사이트, 게시판, 블로그를 만들 수 있다.
    3. 제로보드를 인수(2007년3월14일)한 NHN은 이후 제로보드(zb4.1->zb5)의 개발 및 지원을 중단하고 제로님(고영수)과 프로젝트 멤버를 구성하여 핵심 코어(Core) 프로그램과 모듈형 구조라는 새로운 프레임워크(framework)의웹 CMS(Content Management System) 기능을 갖춘 제로보드XE를 개발하고 배포하였습니다. 이후 제로보드XE는 '보드'의 개념을 벗어나 LGPL 기반 오픈 프로젝트로서 정식 글로벌 CMS 웹빌더라는 의미의 Xpress Engine으로 이름을 변경하고 현재까지 지속적으로 개발이 진행되는 국내 CMS 점유율 1위의 오픈소스 소프트웨어입니다.
    4. XE에는 모듈과 비슷한 기능을 가진 위젯, 애드온, 에디터, 레이아웃, 스킨(skin)
    ※ 가입형 블로그 : 네이버, 야후, 다음, 티스토리
    ※ 설치형 블로그 : 텍스트큐브, 워드프레스

    설치형 워드프레스 시작하기

    설치와 접속

    1. 도메인 구입. 도메인 등록 대행자, hosting.kr
      (이후 설명은 domain.kr 을 구입했다고 가정한다)
    2. 호스팅 세팅. cafe24. DBName / SQL user ID / FTP password
    3. 워드프레스 설치.
      1. http://wordpress.org/ 에서 영문버전(오류 확률 낮음) 다운로드.
      2. wordpress-3.5.1.zip 압축 풀면 /wordpress/ 폴더 자동 생성된다.
      3. FTP 접속. 기본 폴더 (cafe24 는 /www/ 임. 다른 곳은 /public_html/ 등..) 에..
        /wordpress/ 폴더의 "하위 폴더/파일" 을 모두 업로드 : http://domain.kr/ 가 루트가 된다.
        /wordpress/ 폴더를 모두 업로드 : http://domain.kr/wordpress/ 가 루트가 된다.
      4. Web 접속. .. [Create a Configuration File] 클릭! [Les's go!] 클릭!
        Database Name : // 입력
        User Name : // 입력
        Password : // 입력
        Database Host : localhost // 수정할 필요 없음.
        Table Prefix : wp_main // 하나의 계정에 워드프레스 둘 이상 설치한 경우 아니면 그대로 둠.
        [Submit] 클릭!
      5. [Run the install] 클릭! (혹시 mySQL 버전 문제 발생하면 호스팅업체에 문의. mysql 4.x 에서 5.x로 변환신청필요)
        Site Title : // 입력
        Username : // 입력
        Password, twice : // 입력
        Your E-mail : // 입력
        [Install WordPress] 클릭!
      6. Success! ▷ wordpress 로그인 아이디와 패스워드 설정
        Username : // 입력
        Password : // 입력
        [Log In] 클릭!
    4. 워드프레스 접속.
      로그인하면 메인 Dashboard 화면 된다. 앞으로 모든 작업은 Dashboard 를 통해서 한다.
      웹주소가 domain.kr 이면 로그인 주소는 http://domain.kr/wp-login.php 이다.

    테마(스킨) 설정

    1. 테마의 종류도 다양하고 유료에서부터 무료까지 선택의 폭도 넓습니다.
    2. 스킨 쇼핑
      1. 워드프레스 공식 테마 사이트 http://wordpress.org/extend/themes/
      2. 무료 테마 사이트 http://wordpressthemesbase.com/
      3. best of best http://www.wpthemespot.com/
      3. 해당 테마의 zip 파일을 다운받아서 압축 풀지 않고 그대로 보관
    3. Dashboard > Appearance > Themes > Install Themes > Upload 클릭!
    4. ZIP화일을 통째로 선택하여 [Install Now] 클릭!
    5. Active 클릭하여 테마 설치 완료
    6. 테마 설정 : Dashboard > Appereance 항목 조절
    7. 서로 연동되는 테마가 있다. 예) autofocus pro 테마는 Thematic 0.9.7.7 를 먼저 설치 후 다시 활성화해야 한다.
