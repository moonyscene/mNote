    viewport

    <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=yes, target-densitydpi=device-dpi" /> <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    width
    화면 가로 사이트 (device-width로 기기 크기 지정)
    height
    화면 가로 사이트 (device-height로 기기 크기 지정)
    initial-scale
    초기 확대 비율 (0.25~10.0)
    maximum-scale
    최대 확대 비율
    minimum-scale
    최소 확대 비율
    user-scalable
    사용자 Zoom 기능 사용 여부 (기본값 yes | no 이면 Zoom 기능 disable)
    target-densitydpi
    DPI 지정 ( device-dpi 기기 DPI 설정 | medium-dpi )
    Android 기기 DPI 맞춤 렌더링. iOS와 Windows 폰은 해당없음.
    대부분의 모바일 브라우저에서 비슷하려면 값 지정마세요.

    media query

    
/* small~xlarge */
body { min-width: 300px; margin: 0 auto; padding: 0; }

@media all and (min-width:640px) and (max-width:767px) { /* medium~ */
	body { max-width: 604px; padding: 10px; }
}
@media all and (min-width:768px) { /* medium~ */
	body { max-width: 712px; padding: 20px; }
}
@media all and (min-width:1000px) { /* large~ */
	body { max-width: 944px; padding: 20px; }
}
@media all and (min-width:1200px) { /* xlarge~ */
	body { max-width: 1144px; padding: 20px; }
}
@media all and (min-width:1256px) { /* xlarge+(padding20*2)+(scrollbar-track16)~ */
	body { max-width: 1200px; padding: 20px; }
}
@media screen and (orientation:landscape), (min-width:640px) { /* 가로모드 + 폭640px이상 */
	body { padding: 10px; }
}
