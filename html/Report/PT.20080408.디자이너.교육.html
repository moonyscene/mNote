05. 유용한 JavaScript 함수 : myshin.js

  • initImgMenuOn();//여러 페이지에 메뉴를 인클루드해서 사용할 때, 현재 페이지 메뉴만 활성이미지로 표시
  • initImgMenuOnly();//이미지메뉴활성&하나만보이기
  • 목록클릭이벤트할당
    initClickOn("메뉴그룹id");//한페이지, 메뉴그룹안에 콘텐츠그룹 포함시
    initClickOn("메뉴그룹id","활성메뉴id");//여러페이지
    initClickOn("메뉴그룹id","활성메뉴id","콘텐츠그룹id");//한페이지, 메뉴그룹과 콘텐츠그룹 별도시.
