    Photoshop

    1. Palette Character, Paragraph 에서 설정값을 그림 위 마우스 클릭 드래그로 조절할 수 있다.
    2. 포토샵에서 모든 텍스트를 줄 단위로 입력하여 줄간격을 0 으로 하면 잘림 없는 최적의 이미지 커팅 크기를 구할 수 있다.
    3. 포토샵CS에서 Action 으로 제목 이미지 크기가 다른거 저장할 때.. 새 창 크기가 기록 당시로 고정되어 버리는 문제
      다른 버전 테스트 못해봄. 하지만, 해결책은 기록시 새 창 크기글 이미지 최대 보다 더 크게 만들고, 매직 선택툴 [W] 로 빈공간을 찍어서 삭제하고, 다시 이미지 부분만 선택하여, 저장하면 된다. 과제) 너비는 가변이고 높이는 고정 크기로 할 수 없을까?
    4. PSD 제목 이미지 : 나눔고딕 Bold 22px Sharp Metrics -100
    5. 포토샵 원 그릴 때 깔끔한 순 : 도형 툴 > 선택 툴 선그리기 > 선택 툴 색 채우기
