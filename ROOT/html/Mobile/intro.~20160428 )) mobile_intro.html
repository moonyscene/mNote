    viewport

    <meta name="viewport" content="width=device-width,initial-scale=1.0,maximum-scale=1.0,minimum-scale=1.0,user-scalable=yes,target-densityDpi=device-dpi" />
    • width : 화면 가로 사이트 (device-width로 기기 크기 지정)
    • height : 화면 가로 사이트 (device-height로 기기 크기 지정)
    • initial-scale : 초기 확대 비율 (0.25~10.0)
    • maximum-scale : 최대 확대 비율
    • minimum-scale : 최소 확대 비율
    • user-scalable : 사용자가 Zoom기능 사용 제어 (YES | no일 경우 Zoom기능이 disable)
    • target-densityDpi : DPI를 지정(device-dpi로 기기 DPI 설정, medium-dpi?)
