Chart Demo v0.2

Tool / Skill

  1. Chart.js [canvas] ( v2.9.4 [IE11](O) 이용 || 버전 3부터 IE 지원이 중단 )
    데이터 시각화를 위한 무료 오픈 소스 JavaScript 라이브러리로 막대, 선, 영역, 파이, 버블, 레이더, 극좌표 및 산포의 8 가지 차트 유형을 지원합니다.
  2. D3.js [svg] ( v4 [IE9+](O) || v5+ [IE](X) )
    with. Animated bar chart - jQuery plugin : 멀티막대그래프
  3. [svg+css+js] : 도넛 프로그레스
  4. [html+css] : 막대그래프

Plugin

  1. 자동 색상 구성표 적용
    chartjs-plugin-colorschemes :: Color Chart | GitHub
    Chart.js에 대해 미리 정의된 색 구성표입니다. 데이터 시각화를 위한 완벽한 색상 조합을 선택하십시오.
  2. 배경 패턴
    patternomaly.js
    CDN: https://cdnjs.cloudflare.com/ajax/libs/patternomaly/1.3.2/patternomaly.min.js
    데이터 그래픽에 사용할 패턴을 쉽게 생성합니다.
