품질마크 사전설문지 작성 요령

===== 웹 접근성 품질마크 사전설문지는 =====

심사자의 편의를 위해서 제공하는 것뿐입니다.
얼마나 정확하게 잘 작성했는지가 심사 대상이 아닙니다.
품질마크 획득이 목적이라면 접근성이 우수한 콘텐츠만 적으셔도 됩니다.
(그렇다고 해서 그것만 심사하지는 않습니다.)

===== 작성요령 =====

□ 사이트 이용현황
==> 접근성이 우수한 웹페이지만 나열하셔도 됩니다.

□ 로그인 서비스 영역
==> 웹사이트의 내용은 대부분 로그인 없이 볼 수 있으므로 ② 아니오 ( ○ ) 로 하셔도 됩니다.
로그인이 필요한 my○○ 서비스는
접근성 문제가 있는 커뮤니티(동호회, 홈페이지마법사, 웹메일 등)로의 링크를 제공하므로 
심사 대상에 포함시키는건 좋지 않습니다.

□ 멀티미디어 및 기타파일 형식
==> 모든 콘텐츠를 다 기록하실 필요는 없습니다. (원본 양식에도 이렇게 되어 있네요.)
대표적인(웹 접근성이 우수한) 것 1개씩만 적으셔도 됩니다.

□ 품질마크 범위
==> 제외 서비스(부가 사이트)는
신청 사이트의 하위 폴더, 하위 도메인 중 웹 접근성이 부족한 것들을 모두 기재하시면 됩니다.
신청 사이트 외의 부가 사이트는 모두 제외하는게 품질마크 획득에 도움이 되며,
너무 많을 경우 신청 사이트 외는 모두 제외시킨다고 명시해주시면 됩니다.
