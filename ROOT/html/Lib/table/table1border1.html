caption 안에 <span class="tar">(단위:원)</span>사용시 caption 표제목에 공백(&nbsp;)이라도 있어야 제대로 보인다.

  • 열너비 width 는 col 태그에 style 속성으로 지정하며 %값 사용.
  • 셀(td, td)이나 열(col)의 정렬은 각각의 셀(td, td)에 class 로 지정
  • tbdoy 안에 td의 text-align 은 left 가 기본인데, 모두 중앙, 우측 정렬할 때.
    • <tbody class="tdtac">
    • <tbody class="tdtar">
  • 셀(th, td)이나 열(col)의 정렬은 각각의 셀(th, td)에 class 로 지정.
