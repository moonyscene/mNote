정의 목록 불릿 1,2,3,4차

정의 목록 용어 Definition Term
정의 목록 설명 Definition Description. 정의 목록(definition list)은, 항목이 용어(term)와 설명(description) 두 부분으로 구성되어있다는 것 이외에는, 다른 두 종류의 목록과 거의 같다. 용어는 DT 엘레멘트로 지정 되는데 인라인(inline) 내용 만을 가질 수 있고, 내용(description)은 DD 엘레멘트로 지정 되는데 블럭레벨(block-level)의 내용을 가질 수 있다.
메타 데이터 meta data
웹 검색 엔진이나 사이트 관리에 유용한 정보로 메타 데이터를 지정하면 문서 헤더에 메타 태그가 삽입됩니다.
키워드 keyword
검색 엔진이 문서를 내용에 따라 분류하는 데 사용됩니다.
디스크립션 description
설명 메타 태그는 문서의 내용을 간단하게 설명하는 데 사용합니다.

P 엘레멘트는 문단(paragraph)을 나타낸다. 이것은 P 차체 및 블럭레벨 엘레멘트를 포함 할 수 없다. 제작자는 내용이 없는 빈 P 엘레멘트를 사용하지 않는 것이 좋다. 사용도구는 빈 P 엘레멘트를 무시하여야 한다.
