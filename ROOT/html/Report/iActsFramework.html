==>@정대석,박태공
** 메뉴 관련해서.. 코딩팀 요청 사항입니다.
- JSP, PHP, ASP 가 동일한 방식으로 구현 필요?
- 기존대로  이렇게 1, 2차 메뉴별 CSS 적용 필요.
- menu.all 에 있는 공용코드(title태그내용, 현재위치, 본문제목이미지) 생성 기능 필요함.
- menu.all 에 있는 제목명, 링크주소, 클릭속성과 title속성값을 개별 또는 묶어서 사용가능하여야 함.
	mTitle[i][j][k][l] = 제목명
	mLink[i][j][k][l] = 링크주소
	mClick[i][j][k][l] = 클릭속성과 title속성값
	mMenu[i][j][k][l] = 메뉴명
	mAnchor[i][j][k][l] = a요소 속성 전체
- 기존 사용 자바스크립트(topmenu.js, sidemenu.js 등) 와의 연계.
- 여러 경우의 수를 반영해야 함.
예를 들어 주메뉴, 좌메뉴 코드 등을 자동 생성할 때..
3차 메뉴가 대부분 좌메뉴에 도출되지만, 특정 2차메뉴 하위의 3차메뉴는 본문 탭메뉴로 들어가야 하는 경우.
주메뉴 1,2차, 좌메뉴 2,3차 메뉴가 텍스트일 경우와 이미지일 경우 등
- 메뉴명 이미지일 때 파일명 명명과 수정 관리 편의 고려 필요
- 수정시 백업본 생성 및 즉시 복구 기능
- 최초 구축시 sitemap.xml 등 텍스트 파일로도 메뉴 처리 가능해야 함
- DB 메뉴 수정하면 sitemap.xml 도 동시에 수정되어야 함.
- DB 메뉴 로긴 상태 중에 sitemap.xml 수정 금지 및 반대의 경우도 금지 필요.
- 우선 테스트 사이트 만들어서 0.1 버전부터 계속 업그레이드 시키고 실제 과업 적용은 1.0 버전 쯤 될 때 적용
- 디자이너, 유지보수팀 의견도 반영 필요.
