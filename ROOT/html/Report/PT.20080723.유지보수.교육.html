[테스트 환경]
* 한국형 웹 콘텐츠 지침(KWCAG) 1.0
* 브라우저 호환 (for Windows XP) : IE6, IE7, FF최신버전, Op최신버전, Sa최신버전
* 스크린리더 (on IE6 for Windows XP) : 센스리더PE2.2, 드림보이스6.21 (on IE6)
* 스타일시트, 자바스크립트, 플러그인, 이미지 표시 기능 제거(Disable)후 접근성 고려

01. 공공기관 홈페이지 평가

  • 행정기관 웹사이트 진단 및 컨설팅[행자부]- 기술평가
  • 웹접근성실태조사[정통부]
  • 웹접근성품질마크[한국정보문화진흥원-웹접근성연구소]
  • 전자정부 웹 표준 준수지침 제정 고시

05. 유용한 JavaScript 함수 : /share/js/myshin.js

  • initImgMenuOn();//여러 페이지에 메뉴를 인클루드해서 사용할 때, 현재 페이지 메뉴만 활성이미지로 표시
