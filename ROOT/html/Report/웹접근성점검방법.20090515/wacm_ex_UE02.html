익스플로러 (브라우저) 화면 확인

1-3. 콘텐츠의 시각적 명료성

점검 가능 지표

  • 색상으로 표현된 정보는 색상을 배제하여도 원하는 내용을 전달할 수 있어야 한다.
  • 전경색과 배경색은 충분한 대비를 가지고 있어야 한다.
  • 텍스트는 쉽게 읽을 수 있도록 충분한 크기로 제공되어야 한다.

점검 방법 설명

  • 색상 이외에도 명암이나 패턴으로 콘텐츠 구분이 가능한지 확인.
  • 글자 크기가 12px 이상인지 확인
  • 익스플로러 [보기]-[텍스트크기]로 글자 크기가 변하는지 확인.
※ 색상 대비와 텍스트 크기는 객관적 평가가 곤란하여 심사 지표에서 제외되었다.
- 관련근거: 웹 접근성 국가표준 준수 기술 가이드라인(제정일 2009.3.17)
