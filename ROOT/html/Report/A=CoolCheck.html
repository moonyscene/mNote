결론. A=CoolCheck! 테스트 결과

  1. K-Wah 의 페이지 자동수집과 N-WAX 의 대상 페이지의 결함 내용 확인 기능을 혼합한 웹 접근성 진단도구.
  2. 페이지 자동수집은 무제한 가능했지만, 자동검사는 최대 200페이지만 된다.
  3. K-Wah 처럼 특정 서버에서 자동수집 또는 검사 도중 프로그램이 다운되기도 한다.
  4. 검사항목 24개 중 자동검사되는 항목은 최대 7개 였다.
  5. 수동검사 항목은 검사자의 기술 능력에 좌우된다. (자동검사 항목도 수동으로 내용 확인이 필요하다.)
  6. 개발 구현 완료 후 통합테스트 용도로 활용하면 도움되겠다.
  7. 운영서버에서 페이지 자동수집 하다가는 차단될 수 있다.
