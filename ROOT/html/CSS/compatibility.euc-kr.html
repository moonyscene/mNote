font-size/line-height Test

font-size:12px 일 때 line-height 값 없을 때 렌더링 확인해보면 12px~14px 로 표시되어 브라우저별로 다르다.

Result : [IE6,7,8,FF,SF,CR,OP] 동일한 렌더링을 얻으려면?

  • eur-kr (font-size+3)/(font-size) = 15/12 ==> 1.2500..소수점이하2째자리+0.01 = 1.26(line-height)
  • eur-kr (font-size+3)/(font-size) = 14/11 ==> 1.2727..소수점이하2째자리+0.01 = 1.28(line-height)
  • 다른 font-size 도 마찬가지로 환산된 font-size px값 만큼 line-height 값을 변경하면 된다.
  • ※ 단, input, button 요소에서는 다르게 렌더링 된다.
