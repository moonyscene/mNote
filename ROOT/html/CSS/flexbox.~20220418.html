※ 이거보시오! ))) flex - MDN -

축약 속성 { flex : flex-grow flex-shrink flex-basis; } 기본값 flex:0 1 auto; 이고, flex-shrink 와 flex-basis 값은 선택사항이다.


.item { flex: 0 1 auto } /* 기본값 flex: initial; ( flex-grow | flex-shrink | flex-basis ) */
.item { flex: auto } /* flex: 1 1 auto; */
.item { flex: none } /* flex: 0 0 auto; */
.item { flex: 2 } /* flex: 2 1 0; */
.item { flex: 30% } /* flex: 1 1 30%; */
.item { flex: 2 2 } /* flex: 2 2 0; */
.item { flex: 2 30% } /* flex: 2 1 30%; */
.item { flex: 2 2 30% } /* flex-grow | flex-shrink | flex-basis */
