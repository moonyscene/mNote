String Method
Method exam Description
charAt() 문자열.charAt(정수)
정수: 0에서 문자열.length-1까지
 지정된 인덱스에 있는 문자를 반환하여 줍니다.
charCodeAt() 문자열.charCodeAt(문자의 위치) Returns the encoding of the character at a given position in a string.
문자열의 특정 위치에 있는 문자의 유니코드 값을 알아내어 줍니다.
concat() 문자열.concat(문자) Concatenates one or more values to a string.
문자열과 매개변수의 문자를 하나의 문자열로 만들어 줍니다
indexOf() 문자열.indexOf(검색할 문자열, [숫자])
숫자는 검색을 시작할 위치 (0에서 stringName.length-1까지)
 Searches the string for a character or substring.
앞에서부터 검색한 문자열이 처음으로 나타나는 위치를 반환. 찾지 못하면 -1을 반환.
lastIndexOf() 문자열.lastIndexOf(검색할 문자열, [숫자])
숫자는 검색을 시작할 위치 (0에서 stringName.length-1까지)
 Searches the string backward for a character or substring.
뒤에서부터 검색한 문자열이 처음으로 나타나는 위치를 반환. 찾지 못하면 -1을 반환.
match() 문자열.match(정규식) Performs pattern matching with a regular expression.
문자열에서 정규식 검색하여 배열을 반환하여 줍니다.
<a href="#pop1open" onclick="displayOff('pop1'); document.getElementById(this.href.match(/#(.*)/)[1]).focus(); return false;">닫기</a>
replace() 문자열.replace(정규식, 치환할 문자수)
문자열객체.replace(정규식, 치환 후 실행 함수)
 Performs a search-and-replace operation with a regular expression.
정규식에 의한 문자열 치환. 정규식에 의한 문자열 치환 후 함수를 실행합니다.
search() 문자열.search(정규식) Searches a string for a substring that matches a regular expression.
정규식과 일치하는 문자열에 대해 검색하여 위치를 반환. 없으면 -1을 반환.
slice() 문자열.slice(추출시작인덱스, 끝인덱스) Returns a slice or substring of a string.
문자열이나 배열의 일부를 추출합니다.
substring()과의 차이점은 끝인덱스를 음수로 하면 문자열의 끝에서의 위치를 지정할 수 있는 것이다.
split() 문자열.spilt([A],[B])
A: 문자열 분리문자나 기호 또는 정규식
B: 분리할 수
 Splits a string into an array of strings, breaking at a specified delimiter string or regular expression.
문자열을 분리시킵니다.
substr() 문자열.substr(추출시작인덱스,추출문자수) Extracts a substring of a string; a variant of substring().
문자열 일부를 추출.
substring()과 차이점은 추출시작인덱스를 음수로 하면 문자열의 끝에서 부터도 찾을 수 있는 것이다.
substring() 문자열.substring(추출시작인덱스,끝인덱스)
 Extracts a substring of a string.
문자열 일부를 추출. 시작인덱스 포함하여 끝인덱스-1 까지 추출한다. 끝인덱스는 비포함.
시작과 끝인덱스가 같은 값이면 빈 문자열을 반환.
toLowerCase() 문자열.toLowerCase() Returns a copy of the string, with all characters converted to lowercase.
문자열을 소문자로 변환합니다.
toString() 객체이름.toString([radix]) Returns the primitive string value.
객체를 문자열로 바꾸어 줍니다.
toUpperCase() 문자열.toUpperCase() Returns a copy of the string, with all characters converted to uppercase.
문자열을 대문자로 변환합니다.
valueOf() 문자열.valueOf() Returns the primitive string value.
원시 문자열 값을 반환.

// 정적 메소드 사용
var start = Date.now();

doSomethingForALongTime();

var elapsed = end - start; // 밀리초(seconds) 단위의 시간
