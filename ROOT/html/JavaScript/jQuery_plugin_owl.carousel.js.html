    Events

    이벤트는 Owl Carousel에서 전략 코드 위치에 제공됩니다. 이를 통해 변경 사항을 수신하고 자신의 작업을 수행 할 수 있습니다.

    
var owl = $('.owl-carousel');
owl.owlCarousel();
// Listen to owl events:
owl.on('changed.owl.carousel', function(event) {
    ...
})

    올빼미 회전 목마를 제어하기 위해 스스로 이벤트를 실행할 수도 있습니다.

    
var owl = $('.owl-carousel');
owl.owlCarousel();
// Go to the next item
$('.customNextBtn').click(function() {
    owl.trigger('next.owl.carousel');
})
// Go to the previous item
$('.customPrevBtn').click(function() {
    // With optional speed parameter
    // Parameters has to be in square bracket '[]'
    owl.trigger('prev.owl.carousel', [300]);
})

    Callbacks

    이벤트 핸들러를 연결하는 대신 Owl Carousel의 옵션에 콜백을 추가 할 수도 있습니다.

    
$('.owl-carousel').owlCarousel({
    onDragged: callback
});
function callback(event) {
    ...
}

    Data

    각 이벤트는 이벤트 객체 내에서 매우 유용한 정보를 전달합니다. 위의 예를 기반으로 :

    
function callback(event) {
    // 코어에 의해 제공됨
    var element   = event.target;         // DOM 요소 .owl-carousel
    var name      = event.type;           // 이벤트 이름
    var namespace = event.namespace;      // 이벤트 네임 스페이스 owl.carousel
    var items     = event.item.count;     // 항목 수
    var item      = event.item.index;     // 현재 항목의 위치
    // 탐색 플러그인에서 제공됨
    var pages     = event.page.count;     // 페이지 수
    var page      = event.page.index;     // 현재 페이지의 위치
    var size      = event.page.size;      // 페이지 당 항목 수
}

    Carousel

    ★표제목 : ★표요약,구조,탐색방법
    Option Type: Note
    initialize.owl.carousel attachable Callback: onInitialize 플러그인이 초기화 시작하면
    initialized.owl.carousel attachable Callback: onInitialized 플러그인이 초기화된 종료하면
    resize.owl.carousel attachable Callback: onResize 플러그인 크기가 조정 시작하면
    resized.owl.carousel attachable Callback: onResized 플러그인 크기가 조정 종료하면
    refresh.owl.carousel attachable, cancelable, triggerable Callback: onRefresh
    Parameter: [event, speed]    		 플러그인의 내부 상태가 갱신 시작하면
    refreshed.owl.carousel attachable Callback: onRefreshed 플러그인의 내부 상태가 갱신 종료하면
    [m] 로드 또는 리사이즈 하여 너비 갱신된 후, 쪽당 항목 수 변경된 후..
    [m] initialized, resized 와 동시 발생
    drag.owl.carousel attachable Callback: onDrag 드래그 시작하면
    dragged.owl.carousel attachable Callback: onDragged 드래그 종료하면
    translate.owl.carousel attachable Callback: onTranslate 무대 전환이 시작되면
    translated.owl.carousel attachable Callback: onTranslated 무대 전환이 종료되면
    [m] 아이템 이동 후
    change.owl.carousel attachable Callback: onChange
    Parameter: property    		 속성값 변경 시작하면
    changed.owl.carousel attachable Callback: onChanged
    Parameter: property    		 속성값 변경 종료하면
    [m] 로드 + 리사이즈 + 아이템 이동 후 모두 발생
    [m] initialized, resized 와 동시 발생 translated 이전에 발생
    next.owl.carousel triggerable Parameter: [speed] 다음 항목으로 이동
    prev.owl.carousel triggerable Parameter: [speed] 이전 항목으로 이동
    to.owl.carousel triggerable Parameter: [position, speed, boolean] 인덱스 위치로 이동 
    
// 아이템 위치로 이동
$('.goItem .to-m', $my).on('click', function(event){
  $owl.trigger( 'to.owl.carousel', [$(this).index(), , true] );
});

// 페이지 위치로 이동
$('.goPage .to-m', $my).on('click', function(event){
  $owl.trigger( 'to.owl.carousel', [$(this).index()] );
});
    destroy.owl.carousel triggerable Parameter: 회전목마를 파괴
    replace.owl.carousel triggerable Parameter: data 현재 내용을 제거하고 매개 변수에 전달 된 새 내용을 추가
    add.owl.carousel triggerable Parameter: [data, position] 주어진 위치에 새 항목을 추가
    remove.owl.carousel triggerable Parameter: position 지정된 위치에서 항목을 제거
