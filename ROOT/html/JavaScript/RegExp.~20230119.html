Regular Expression

Create Regular Expression

var re = /패턴/플래그; var re = new RegExp("패턴","플래그"); (차이) new로 만들때에는 이스케이프문자는 \는 \\로 해주어야 한다. var re = /\w/; var re = new RegExp("\\w");

Method

Regular Expression Method

: ★표요약,구조,탐색방법
Object Method Note
RegExp RegExp.exec(String) 문자열을 검색해서 매치되면 배열을 반환하고, 아니면 null을 반환
RegExp.test(String) 문자열을 검색해서 매치되면 true, 아니면 false를 리턴한다.
String String.match(RegExp) 문자열을 검색해서 매치되면 배열을 반환하고, 아니면 null 을 반환
String.replace(RegExp, replacement) 문자열을 검색해서 매치된 문자열을 치환
String.search(RegExp) 문자열을 검색해서 매치된 인덱스를 반환, 매치되지 않으면 -1을 반환
String.split(RegExp) 정규 표현 또는 문자열을 사용하여 검색 대상 문자열을 분할하여 배열을 반환

Flag

flag Note
i (Ignore Case) 대소문자 무시
g (Global) 전체.
여러줄이면.. /^/ 1개, /\n/ 줄수-1, /$/ 1개
m (Multiline) 여러줄. g 와 같이 사용안하면 1줄만 찾음.
여러줄이면.. /^/ 1개, /$/ 1개, /\n/ 1개
gm 또는 mg (Global Multiline) 전체 여러줄.
여러줄이면.. /^/ 줄수, /$/ 줄수, /\n/ 줄수-1

Pattern

pattern Note
^ 문자열의 시작을 의미 ,m 플래그를 사용할경우 경우는 각 문자열의 시작
$ 문자열의 끝을 의미 ,m 플래그를 사용할경우 경우는 각 문자열의 끝
. 모든 한문자
[문자들] 괄호안의 문자 하나와 매치
예) [abc] 는 a나 b나 c중 하나를 의미
[^문자들] 괄호안의 문자가 아닌문자와 매치
예) [^abc] 는 1,2.... d,e.... 등과 매치
[문자1-문자2]
 문자1과 문자2와 그 사이의 값과 매치
예) [a-d] a,b,c,d와 매치
(abc) abc와 매치
| 좌우 패턴중 하나를 의미
예) (abc|def) abc나 def를 의미
* 앞의 패턴이 0회 또는 그 이상반복됨
+ 앞의 패턴이 1회 또는 그 이상반복됨
? 앞의 패턴이 0또는 1회 반복
{n}, {n,}, {n,m} 패턴의 반복회수
예)
(abc){1,3} abc가 1에서 3회 반복
(abc){1} abc가 1회반복
(abc){,10} abc가 10회 이하 반복

특수문자 (Escapes Character)

Escapes Character Note
\ 일반문자에 \을 붙여서 특수한 용도로 사용한다.
\f 폼피드(?)
\r 캐리지리턴
\n 새줄
\t 일반 탭문자
\v 세로 탭문자(?)
\0 NUL널문자
[\b] 백스페이스
\s 공백문자. [ \t\n\r\f] 와 같다. \t\n\r\f 를 제외한 공백은 [ ] 다.
\S 공백이 아닌 문자. [^ \t\n\r\f] 와 같다.
\w 숫자,알파벳,_(언더바) 문자. [0-9A-Za-z_] 와 같다.
\W 숫자,알파벳,_(언더바) 가 아닌 문자. [^0-9A-Za-z_] 와 같다.
\d 정수. [0-9] 와 같다.
\D 정수가 아닌 문자. [^0-9] 와 같다.
\b 단어의 경계 공백, 새줄.
\B 경계가 아닌 문자.
\cX 컨트롤+문자 E.g: \cm matches control-M.
\xhh 핵사코드
\uhhhh 유니코드
\f, \n, \r, \t, \v, \u00A0, \u2028, \u2029

매치된 문자열

$1, $2, $3, .. !주의! $0 은 없다. 매치된 전체 문자열은 '$&' 를 사용하거나 정규표현식 전부를 ()로 묶어서 '$1' 를 사용하라! str2=str2.replace(/(mAll\[[0-9]+\]\[0\]\[0\]\[0\])/gm,'\n//1depth\n$1');

Example

var filter = /a/ // 'a' 가 있는 문자열이면 TRUE (대소문자 구분) if (filter.test("some test words") == true ) { // 테스트 샘플 alert("ok"); } else { alert("fail"); } var filter = /about/ // "about" 가 있는 문자열 모두가 TRUE (대소문자 구분) var filter = /a/i // 'a' 또는 'A' 가 있는 문자열 모두가 TRUE (대소문자 구분 안함) var filter = /[a-z]/ // 'a' 에서 'z' 까지중 하나만 있으면 모두가 TRUE (대소문자 구분) var filter = /a|b|c/ // 'a' 또는 'b' 또는 'c' 가 있는 문자열이면 TRUE (대소문자 구분) var filter = /[a-z]|[0-9]/ // 'a' ~ 'z' 또는 '0' ~ '9' 중 하나만 있으면 TRUE (대소문자 구분)

[^문자열] 문자(열)이 없는 경우. bracket [] 안에 ^

var filter = /[^a-z]/ // 'a' 에서 'z' 까지의 문자가 아닌 문자가 있을 경우 TRUE (대소문자 구분)

^[문자열] 문자열로 줄시작하는 경우. bracket [] 밖에 ^

var filter = /^[a-z]/ // 'a' ~ 'z' 문자로 시작하는 문자열이면 TRUE (대소문자 구분) var filter = /[a-z]$/ // 'a' ~ 'z' 까지의 문자로 끝나는 문자열일 겨우 TRUE (대소문자 구분)

특수문자 \ ^ $ * + ? . ( ) | { } [ ] 를 검색하려면 앞에 \

var filter = /\\/ // '\' 가 있는 문자열일 겨우 TRUE (대소문자 구분)
