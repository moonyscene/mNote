정수: 0에서 문자열.length-1까지
 지정된 인덱스에 있는 문자를 반환하여 줍니다.

concat() 문자열.concat(문자) 문자열과 매개변수의 문자를 하나의 문자열로 만들어 줍니다.
indexOf() 문자열.indexOf(검색할 문자열, [숫자])
숫자는 검색을 시작할 위치 (0에서 stringName.length-1까지)
 앞에서부터 검색한 문자열이 처음으로 나타나는 위치를 반환. 찾지 못하면 -1을 반환.
lastIndexOf() 문자열.lastIndexOf(검색할 문자열, [숫자])
숫자는 검색을 시작할 위치 (0에서 stringName.length-1까지)
 뒤에서부터 검색한 문자열이 처음으로 나타나는 위치를 반환. 찾지 못하면 -1을 반환.
match() 문자열.match(정규식) 문자열에서 정규식 검색하여 배열을 반환하여 줍니다.

문자열객체.replace(정규식, 치환 후 실행 함수)
 정규식에 의한 문자열 치환. 정규식에 의한 문자열 치환 후 함수를 실행합니다.
search() 문자열.search(정규식) 정규식과 일치하는 문자열에 대해 검색하여 위치를 반환. 없으면 -1을 반환.
slice() 문자열.slice(추출시작인덱스, 끝인덱스) 문자열이나 배열의 일부를 추출합니다.
substring()과의 차이점은.. 끝인덱스를 음수로 하면 문자열의 끝에서의 위치를 지정할 수 있는 것이다.
split() 문자열.spilt([A],[B])
A: 문자열 분리문자나 기호 또는 정규식
B: 분리할 수
 문자열을 문자열 배열로 분리하여 반환한다. 지정된 구분 기호 문자열이나 정규 표현식으로 분리합니다.
substr() 문자열.substr(추출시작인덱스,추출문자수) 문자열 일부를 추출.
substring()과 차이점은.. 추출시작인덱스를 음수로 하면 문자열의 끝에서부터 찾는다.
substring() 문자열.substring(추출시작인덱스,끝인덱스)
 문자열 일부를 추출. 시작인덱스 포함하여 끝인덱스-1 까지 추출한다. 끝인덱스는 비포함.
시작과 끝인덱스가 같은 값이면 빈 문자열을 반환.
toLowerCase() 문자열.toLowerCase() 문자열을 소문자로 변환합니다.
toString() 객체이름.toString([radix]) 객체를 문자열로 변환합니다.
toUpperCase() 문자열.toUpperCase() 문자열을 대문자로 변환합니다.
valueOf() 문자열.valueOf() 원시 문자열 값을 반환합니다.
