Top 5 Tips for Making Fast RWD Sites

  1. Avoid downloading hidden images by using a JavaScript-based or CSS-based image loading technique.
  2. Download images appropriately sized to the relevant screen, a technique known as “Responsive Images“.
  3. Conditionally load JavaScript, and especially 3rd party JavaScript.
  4. Use RESS – REsponsive + Server Side – to optimize your site for known clients.
  5. Introduce performance testing into your QA or build process.

빠른 RWD 사이트를 만들기위한 5 가지 팁

  1. 자스크립트 기반 또는 CSS 기반의 이미지 로딩 기술을 사용하여 숨겨진 이미지를 다운로드 마십시오.
  2. 관련 화면에 적절한 사이즈의 이미지를 다운로드 하시오! "Responsive Images" 로 알려진 기술.
  3. 조건부로 자바스크립트, 특히 서드파티 자바스크립트를 로드하시오!
  4. RESS(REsponsive + Server Side)를 사용하시오! - 알려진 고객을 위해 사이트를 최적화 할 수 있습니다.
  5. 성능 테스트를 소개하시오! 당신의 QA로 소개 또는 프로세스를 구축 할 수 있습니다.
