    Element script must not have attribute defer unless attribute src is also specified.
    (요소 script 는 속성 src 도 지정하지 않은 연기 속성 defer 가 없어야합니다.)
    <script defer="defer">/*<![CDATA[*/ /*]]>*/</script> [m.note] XHTML1.0 Strict 검사하면 오류로 검출하지 않는다. 원래 잘못된 거였으나, 전에는 유효성 검사기가 검출하지 않은 걸로 판단된다.
    아래 코드와 같이 src 속성으로 외부 js 파일을 불러올 때 사용하여야 한다. js 실행을 연기하는 역할을 한다.
    <script src="http://code.jquery.com/jquery-latest.min.js" defer="defer"></script> A script that will not run until after the page has loaded.
    (페이지가 로드 된 후 때까지 스크립트가 실행되지 않습니다.)
