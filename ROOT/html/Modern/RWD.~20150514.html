    Media Query

    • 작은 화면 기본 CSS 를 먼저 작성하고 미디어 쿼리로 큰 화면 CSS 를 높은 우선 순위가 되게 작성한다.
    • 미디어 쿼리 코드는 기본 CSS 파일의 맨 아래에 작성한다.
    • 기본 CSS 파일과 별도 미디어 쿼리 CSS 파일 사이에 JavaScript 등 다른 외부 파일을 불러오지 않는다.
    • only 키워드는 미디어 쿼리 지원 UA 에서 구문을 해석하라는 명령이며 생략 가능하다.
    • 유일 조건 또는 only, all과 같은 선행 키워드가 생략되면 and 키워드는 사용하지 않는다.
    • [m] sceen, print 미디어 종류만 유일한 조건이면 only 는 사용하지 않는다.
      only 를 사용하면.. [~IE8] 미디어 쿼리 지원 js 가 필요한 UA에서 js 로딩 후 css 가 실행되어 완전하지 않는 화면을 잠시 보게 될것이다.

    Media Query 코드 샘플

    /* Media Query */ @media only screen and (orientation:portrait){/* 세로 mode */ /* ..여기에.. css 정의 두면 된다 */ #container{width:480px;} } @media only screen and (orientation:landscape){/* 가로 mode */ } @media only screen and (max-width:320px){/* 320px 이하 */ } @media only screen and (min-width:321px){/* 321px 이상 */ }

    Media Query 파일 분리

    <link rel="Stylesheet" media="screen and (min-width:768px)" type="text/css" href="css/default768.css" /> <link rel="Stylesheet" media="screen and (min-width:1024px)" type="text/css" href="css/default1024.css" />
