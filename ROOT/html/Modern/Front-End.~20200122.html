WebRTC 는 실시간으로 오디오, 비디오, 데이터를 사용하고 하나의 표준화된 프로토콜을 사용해 웹과 연결된 기기가 서로 통신할 수 있도록 하는 것을 목표로 하는 흥미로운 API 집합이다.

어쨌든 현재 그 모습을 정확히 알 수는 없지만 실시간 통신은 웹에서 실현될 것이다.
